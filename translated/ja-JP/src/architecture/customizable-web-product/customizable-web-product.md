# ベースアプリケーションのカスタマイズバージョンの作成

様々なインスタンスによって使用されることが多い製品アプリケーションがある場合、ユースケースに合わせて個別のカスタマイズが必要となる可能性があります。ルックアンドフィールを変更したり簡単なユーザーインタラクションを微調整したりする場合があります。

様々なインストールでのカスタマイズが想定される製品を設計するにはどのようにすればよいですか？

## 回答
